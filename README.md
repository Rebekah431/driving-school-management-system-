# Driving School Management System

## Overview
A Python-based system for managing driving school operations, including instructors, vehicles, and lesson bookings.

## Features
- Add and manage instructors
- Add and manage vehicles
- Create lesson bookings
- Prevent double bookings (conflict detection)
- View all bookings with joined data

## Technologies
- Python
- SQLite

## How to Run
1. Ensure Python is installed
2. Run:
   python app.py

## Key Learning
- Database design and relationships
- SQL queries and joins
- Input handling and validation
- Conflict detection logic

## IT & Support Skills Demonstrated
- Managing structured data systems
- Troubleshooting booking conflicts
- Ensuring system reliability
- Working with relational databases
